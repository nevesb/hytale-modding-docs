## What you'll find here

Static documentation generated from Hytale's configuration files. Built for server mod creators: NPCs, items, recipes, drop tables, and world rules via JSON.

- **Getting Started** — Installation, project structure, JSON basics, and asset overview.
- **Reference** — Concepts, NPC system, items, crafting, combat, economy, world, models, and game configuration.
- **Tutorials** — Beginner to advanced, including showcases like *Dude vs Dungeon* and mod packaging.
