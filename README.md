## Como Usar Este Manual

1. Siga os capitulos em ordem para uma experiencia completa de aprendizado
2. Cada capitulo corresponde a um video da playlist
3. Os capitulos incluem explicacoes detalhadas, exemplos de codigo e referencias

## Pre-requisitos

- Hytale instalado
- Blockbench (para modelagem)
- Editor de texto (VS Code recomendado)
- Conhecimento basico de JSON
